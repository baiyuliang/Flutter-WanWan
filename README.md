# wanwan

A Flutter application.

该项目可以学习到什么？

 - 常见布局方式；
 - 常见控件使用方式；
 - 自定义控件封装方法；
 - 网络请求封装；
 - 即时通讯IM示例（<font color="FF0000"> 已接入融云IM）；
 - 朋友圈视频播放、图片预览示例；
 - 数据库moor；
 - 待续...
